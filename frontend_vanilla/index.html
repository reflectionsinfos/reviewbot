ReviewBot CLI Agent

Download Agent (.zip)

Run autonomous code reviews from your terminal or CI/CD pipeline. Commands below are pre-filled with your selected project and checklist.

1 Install the agent — one time only
pip install reviewbot-agent
2 Login — token lasts 8 hours, no prior auth needed
reviewbot login --server https://reviewbot-web-128263129038.us-central1.run.app
3 Run a review IDs pre-filled from dropdowns above — select a project & checklist above to pre-fill IDs
reviewbot review \ --project-id <project-id> \ --checklist-id <checklist-id> \ --source-path ./your-project \ --output ./review-report.md
Other commands
Check login status
reviewbot status
Logout
reviewbot logout
Token expires after 8 hours. Re-run reviewbot login if you get a 401 Unauthorized error.
Credentials stored at ~/.reviewbot/config.json (Windows: %USERPROFILE%\.reviewbot\config.json).
